Crea tu token de acceso
El token permite que la app guarde cambios en tu repo automáticamente. Es como tu contraseña de GitHub pero solo para este proyecto.
Cómo crear tu token
  1. Haz clic en el botón de abajo
  2. En Note escribe: PLN Cultura Digital
  3. En Expiration elige No expiration
  4. En Scopes activa repo
  5. Clic en Generate token
  6. Copia el token (empieza con ghp_) y pégalo abajo
